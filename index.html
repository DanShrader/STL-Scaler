STL Scale Adjuster

This is a simple single page application to allow for STL adjustments based on how your 3d printer prints. The default values are what my printer produces from a XYZ 20mm Calibration Cube, but you can simply insert your own numbers.

Instructions
  1. Print a calibration object
  2. Enter what the measurment should be
  3. Enter what it is
  4. Upload the STL file
  5. Export the corrected STL file

Warning
Like anything with 3D printing use this at your own risk. It's free software, but that does not mean that it's free of bugs. Check your exported STL files before printing. There is no warrenty or guarentee that it will work, or that the files will be right.
